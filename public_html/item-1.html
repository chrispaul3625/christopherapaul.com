Sprots Project

What is Sports?

Sprots is a dynamic website that allows users to access statistics to various different teams in the NFL, NHL, NBA, and MLB. Users can create an account with us which allows them to favorite their favorite teams and receive updated scores and statistics on those teams, so that they don't have to navigate through all the teams in a league. There are dynamic search options that are built into the page as well allowing users to type in a team or players name that auto sorts through a table of teams or players so that the user can quickly find a player or team.

Technical aspects of the project Sprots

Sprots was created by a team of 5, including myself. We built the entire project from scratch, creating ERD models to help us visualize the project, creating the database to store all of our statistics and information for the website including profiles, writing out classes in PHP, writing code to sanitize all input areas to protect the site from hackers, We wrote code that would reach out to an external API which is where all the statistical data of the site would come from and cache it into our database. We also created wireframes and diagrams for the layout of the pages and finally used Angular JS to make the page a single page application allowing for quick page transitions being that our site has so much data. Lastly we designed the site with CSS and bootstrap.

What's next for Sprots?

Sprots was built as a project in a 10 week long programming class. In this short period of time we were able to accomplish everything talked about above, however the external API that we were getting all our statistical information from moved to a paid subscription that we were not able to continue to keep as the costs were not within our groups budget. We did take the site offline after this. In the future we plan to continue to explore different API sources and also plan to add other sports to the website as well and bring the site live once again. Feel free to check out our work on this project at https://github.com/chrispaul3625/sprots.
